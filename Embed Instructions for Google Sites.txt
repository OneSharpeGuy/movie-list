Embed Instructions for Google Sites – Dynamic Last Updated Display
==================================================================

You are using the dynamic `lastUpdated.html` file to display the timestamp
of the last commit to specific files in your GitHub Pages repository.

Below are the exact URLs to use when embedding this feature into Google Sites.

------------------------------------------------------------------
Embed URL for Top 100 Movies Timestamp
------------------------------------------------------------------
https://onesharpeguy.github.io/movie-list/lastUpdated.html?file=Top100.html

Embed this to show the last updated date for Top100.html.

Instructions:
1. Open your Google Sites editor.
2. Click “Embed” from the Insert panel.
3. Choose the “By URL” tab.
4. Paste the URL above.
5. Click “Insert”.

Optional Styling Notes
------------------------------------------------------------------
